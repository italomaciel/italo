## Olá! Eu sou o Ítalo Maciel 👋



Meu primeiro contato com internet e computador foi nos anos 2000, o olho brilhou. Mas por morar no interior, achei que trabalhar com qualquer coisa relacionada a tecnologia não seria possível.

Alguns anos passaram, o acesso a internet se tornou mais fácil e descobri um mundo de possibilidades, assim como que a geografia não era mais um obstáculo. Comecei a estudar programação por puro interesse. Realizei cursos onlines para desenvolvimento WEB, Smart Contracts, BOTs e desenvolvimento mobile.
